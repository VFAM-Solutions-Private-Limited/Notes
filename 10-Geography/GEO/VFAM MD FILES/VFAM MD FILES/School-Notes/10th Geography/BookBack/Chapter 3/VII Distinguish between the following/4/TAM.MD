### 4. வண்டல் மண் மற்றும் கரிசல் மண்
 
| அம்சம் | **வண்டல் மண்** | **கரிசல் மண்** |
|---|---|---|
| **உருவாக்கம்** | ஆறுகள் மணல், வண்டல் படிவுகளை படியவைப்பதால் | தக்காண பசால்ட் பாறைகள் சிதைவுறுவதால் |
| **நிறம்** | வெளிர் சாம்பல் முதல் அடர் சாம்பல் | அடர் கருப்பு |
| **செழிப்பு** | பொட்டாசியம், சுண்ணாம்பு, பாஸ்போரிக் அமிலம் | கால்சியம், மக்னீசியம், இரும்பு, அலுமினியம் |
| **குறைபாடு** | நைட்ரஜன் குறைவு | நைட்ரஜன், இலைமட்கு குறைவு |
| **நீர்தக்கவைக்கும் திறன்** | நடுத்தர அளவு | மிகவும் அதிகம் |
| **ஈரமாக இருக்கும் போது** | வண்டல் மணலாக இருக்கும் | மிகவும் சேறாக, ஒட்டும் தன்மை |
| **முக்கிய பயிர்கள்** | **நெல், கோதுமை**, கரும்பு, எண்ணெய் வித்துகள் | **பருத்தி**, தினை, கரும்பு, புகையிலை |
| **இடம்** | கங்கை, பிரம்மபுத்திரா சமவெளிகள், கிழக்கு கடற்கரை | மகாராஷ்டிரா, மாளவ பீடபூமி, தெலங்கானா |
| **வேறு பெயர்கள்** | காதர் (புதியது), பாங்கர் (பழையது) | ரேகூர் மண் / பருத்தி மண் |
