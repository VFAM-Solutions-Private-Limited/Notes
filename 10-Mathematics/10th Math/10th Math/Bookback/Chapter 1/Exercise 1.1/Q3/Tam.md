# பயிற்சி 1.1

## கேள்வி 3:

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)} எனில், A மற்றும் B ஆகியவற்றைக் காண்க.

---

## தீர்க்கும் முறை (குறிப்புக்கு):
1. B x A இல் உள்ள ஒவ்வொரு வரிசை இணை (Ordered Pair) யின் முதல் உறுப்புகள் தொகுதி (Set) B-க்கு உரியவை.  
2. B x A இல் உள்ள ஒவ்வொரு வரிசை இணை (Ordered Pair) யின் இரண்டாவது உறுப்புகள் தொகுதி (Set) A-க்கு உரியவை.  
3. ஒரே உறுப்புகள் மீண்டும் வந்தால், அவற்றை ஒருமுறை மட்டும் எடுத்துக்கொள்ளவும்.  
4. பெறப்பட்ட உறுப்புகளை தொகுதி (Set) வடிவில் எழுதவும்.

---

## விடை:

#### கொடுக்கப்பட்டது:

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

நாம் அறிந்தது:

B x A = {(b,a) | b ∈ B மற்றும் a ∈ A}

காண வேண்டியது: A மற்றும் B-ஐ காண்க.

இதன் பொருள்:

- ஒவ்வொரு வரிசை இணையின் முதல் உறுப்பு தொகுதி (Set) B-க்கு சொந்தமானது.  
- ஒவ்வொரு வரிசை இணையின் இரண்டாவது உறுப்பு தொகுதி (Set) A-க்கு சொந்தமானது.

---
#### விடை:

```text
B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

    A = {3,4}

    B = {-2,0,3}
```
---
## மேலும் தெளிவாகப் புரிந்துகொள்ள (Optional):
```text
படி 1: தொகுதி (Set) B-ஐ காண்க

அனைத்து வரிசை இணை (Ordered Pair) களின் முதல் உறுப்புகளை எடுத்தல்:

(-2,3) → -2
(-2,4) → -2
(0,3)  → 0
(0,4)  → 0
(3,3)  → 3
(3,4)  → 3

வேறுபட்ட உறுப்புகள்: -2, 0, 3

எனவே, B = {-2,0,3}


படி 2: தொகுதி (Set) A-ஐ காண்க

அனைத்து வரிசை இணை (Ordered Pair) களின் இரண்டாவது உறுப்புகளை எடுத்தல்:

(-2,3) → 3
(-2,4) → 4
(0,3)  → 3
(0,4)  → 4
(3,3)  → 3
(3,4)  → 4

வேறுபட்ட உறுப்புகள்: 3, 4

எனவே, A = {3,4}


படி 3: சரிபார்த்தல்

இப்போது சரிபார்க்க:

B x A = {-2,0,3} x {3,4}

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

இது கொடுக்கப்பட்ட விடையுடன் பொருந்துகிறது.


இறுதி விடை:

A = {3,4}

B = {-2,0,3}
```
