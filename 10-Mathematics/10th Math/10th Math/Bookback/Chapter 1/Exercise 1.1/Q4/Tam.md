# பயிற்சி

## கேள்வி:

A = {5,6}, B = {4,5,6}, C = {5,6,7} எனில்,  
A x A = (B x B) ∩ (C x C) எனக் காட்டுக.

---

## தீர்க்கும் முறை (குறிப்புக்கு):
1. A தொகுதியின் ஒவ்வொரு உறுப்பையும் அதே A தொகுதியின் அனைத்து உறுப்புகளுடன் ஜோடியாக அமைத்து A x A-ஐ காண்க.  
2. அதே முறையில் B x B-ஐ காண்க.  
3. அதே முறையில் C x C-ஐ காண்க.  
4. (B x B) மற்றும் (C x C) ஆகியவற்றில் பொதுவாக உள்ள வரிசை ஜோடிகளை (Common Ordered Pairs) கண்டறியவும்.  
5. அந்த முடிவை A x A உடன் ஒப்பிடவும்.  
6. இரண்டும் சமமாக இருந்தால், கொடுக்கப்பட்ட சமத்துவம் நிரூபிக்கப்படும்.

---

## விடை:

#### கொடுக்கப்பட்டது:

A = {5,6}  
B = {4,5,6}  
C = {5,6,7}

```text
LHS:

A x A-ஐ காண:

A x A = {5,6} x {5,6}

      = {(5,5), (5,6), (6,5), (6,6)} -->(1)

RHS:

B x B-ஐ காண:

B x B = {4,5,6} x {4,5,6}

      = {(4,4), (4,5), (4,6),
         (5,4), (5,5), (5,6),
         (6,4), (6,5), (6,6)}


C x C-ஐ காண:

C x C = {5,6,7} x {5,6,7}

      = {(5,5), (5,6), (5,7),
         (6,5), (6,6), (6,7),
         (7,5), (7,6), (7,7)}


(B x B) ∩ (C x C)-ஐ காண:

(B x B) ∩ (C x C) = {(5,5), (5,6), (6,5), (6,6)} --->(2)

LHS = RHS

(1) மற்றும் (2) இலிருந்து,

A x A = (B x B) ∩ (C x C)
```
