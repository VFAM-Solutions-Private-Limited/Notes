# பயிற்சி 1.1

## கேள்வி 5:

A = {1,2,3}, B = {2,3,5}, C = {3,4} மற்றும் D = {1,3,5} எனில்,  
(A ∩ C) x (B ∩ D) = (A x B) ∩ (C x D) என்பது உண்மையா என சோதிக்கவும்.

---
## விடை:

#### கொடுக்கப்பட்டது:

A = {1,2,3}  
B = {2,3,5}  
C = {3,4}  
D = {1,3,5}

---

```text
LHS => (A ∩ C) x (B ∩ D)

    A ∩ C = {1,2,3} x {3,4} = {3}

    B ∩ D = {2,3,5} x {1,3,5} = {3,5}

    (A ∩ C) x (B ∩ D) = {3} x {3,5}

                      = {(3,3), (3,5)}...(1)


RHS => (A x B) ∩ (C x D)
 
    A x B = {1,2,3} x {2,3,5} = {(1,2), (1,3), (1,5), (2,2), (2,3), (2,5), (3,2), (3,3), (3,5)}

    C x D = {3,4} x {1,3,5} = {(3,1), (3,3), (3,5), (4,1), (4,3), (4,5)}

    (A x B) ∩ (C x D) = {(1,2), (1,3), (1,5), (2,2), (2,3), (2,5), (3,2), (3,3), (3,5)} x {(3,1), (3,3), (3,5), (4,1), (4,3), (4,5)}

                      = {(3,3), (3,5)}...(2)

LHS = RHS

(1) மற்றும் (2) இலிருந்து,
(A ∩ C) x (B ∩ D) = (A x B) ∩ (C x D)
```
---

## இறுதி விடை:
கொடுக்கப்பட்ட கூற்று உண்மை (TRUE).
