## Question 2:

Let A = {1, 2, 3 ,4, . . . , 45} and R be the relation defined as “square is of a number” on A.<br>
Write R as a subset of A x A.<br>
Also, find the domain and range of R. 

---

## Procedure to solve (For reference):

1. Take each element “a” from set A.  
2. Find the square of that element (a²). 
3. Check if a² is present in set A.  
4. If yes, form the ordered pair (a, a²). 
5. Repeat the process for all elements of A. 
6. Write all such pairs to form relation R.  
7. Domain is all first elements of the pairs.  
8. Range is all second elements of the pairs.

---

## Solution

#### Given:
A = {1, 2, 3, 4, …, 45}  
R: “square of a number”  

---

(a, b) ∈ R if b = a² and both a, b ∈ A.

#### Check squares of elements:

```text
- 1² = 1  
- 2² = 4  
- 3² = 9  
- 4² = 16  
- 5² = 25  
- 6² = 36  
- 7² = 49 (49 is greater 45)

Thus, valid values are:  
a = {1, 2, 3, 4, 5, 6}
```
