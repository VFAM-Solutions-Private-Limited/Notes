# Exercise 1.2  

## Question 3:

A Relation R is given by  
R = { (x, y) | y = x + 3 , x ∈ {0, 1, 2, 3, 4, 5} }  
Determine its domain and range.

---

## Procedure to solve (For reference):

1. Take each element “x” from the given set.  
2. Apply the relation formula (y = x + 3).  
3. Find the value of y.  
4. Form ordered pairs (x, y).  
5. Write all such pairs to form relation R.  
6. Domain is all first elements.  
7. Range is all second elements.  

---

## Solution

#### Given:
x ∈ {0, 1, 2, 3, 4, 5}  
R: y = x + 3  

---

#### To find values:

```text
Substitute each value of x into y = x + 3:

- When x = 0 → y = 0 + 3 = 3 → (0, 3)  
- When x = 1 → y = 1 + 3 = 4 → (1, 4)  
- When x = 2 → y = 2 + 3 = 5 → (2, 5)  
- When x = 3 → y = 3 + 3 = 6 → (3, 6)  
- When x = 4 → y = 4 + 3 = 7 → (4, 7)  
- When x = 5 → y = 5 + 3 = 8 → (5, 8)  
```
