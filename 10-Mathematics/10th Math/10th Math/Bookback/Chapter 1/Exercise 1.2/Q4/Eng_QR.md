# Exercise 1.2  

## Question 4:

Represent each of the given relations by  
(a) an arrow diagram, (b) a graph and (c) a set in roster form, wherever possible.

(i) R = { (x, y) | x = 2y , x ∈ {2, 3, 4, 5}, y ∈ {1, 2, 3, 4} }<br>
(ii) R = { (x, y) | y = x + 3 , x, y are natural numbers < 10 }

---

## Solution

#### (i) Given:
x ∈ {2, 3, 4, 5}  
y ∈ {1, 2, 3, 4}  
R: x = 2y  

---

#### To find values:


```text
x = 2y → y = x/2

Substitute each value of x into y = x/2:

- When x = 2 → y = 2/2 = 1 → (2,1) 
- When y = 3 → y = 3/2 → (3,3/2)
- When y = 4 → y = 4/2 = 2 → (4,2) 
- When y = 5 → y = 5/2 → (5,5/2) 

R = {(2,1), (3,3/2),(4,2), (5,5/2)}
```

#### (ii) Given:
x, y are natural numbers < 10  
R: y = x + 3  

---

#### To find values:

```text
Substitute each value of x into y = x + 3:

- When x = 1 → y = 1 + 3 = 4 → (1,4)  
- When x = 2 → y = 2 + 3 = 5 → (2,5)  
- When x = 3 → y = 3 + 3 = 6 → (3,6)  
- When x = 4 → y = 4 + 3 = 7 → (4,7)  
- When x = 5 → y = 5 + 3 = 8 → (5,8)  
- When x = 6 → y = 6 + 3 = 9 → (6,9)  
- When x = 7 → y = 7 + 3 = 10 → (7,10) 
- When x = 8 → y = 8 + 3 = 11 → (8,11)
- When x = 9 → y = 9 + 3 = 12 → (9,12)

R = {(1,4), (2,5), (3,6), (4,7), (5,8), (6,9), (7,10), (8,11), (9,12)}
```
