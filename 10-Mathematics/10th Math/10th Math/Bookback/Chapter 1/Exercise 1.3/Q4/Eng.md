## Procedure to solve (For reference):

1. Read the corresponding y-value for each given x-value from the graph.
2. To find x when f(x) = 1, locate y = 1 on the graph and find the corresponding x-value.
3. The Domain is the set of all x-values for which the function is defined.
4. The Range is the set of all y-values taken by the function.

---

## Solution:



#### (i) Values of the function
From the graph,

```text
(a) f(0)  = 9
(b) f(7)  = 6
(c) f(2)  = 6
(d) f(10) = 0
```

---

#### (ii) For what value of x is f(x) = 1?

From the graph,

```text
f(x) = 1 when x = 9.5
```

---

#### (iii) Domain and Range

#### Domain

The graph is defined from x = 0 to x = 10.

```text
Domain = [0,1,2,3,4,...,10]

       = {x/0 ≤ x ≤ 10, x ∈ R}
```

#### Range

The minimum value of f(x) is 0 and the maximum value is 9.

```text
Range = [0,1,2,3,...,9]

      = {x/0 ≤ x ≤ 9, x ∈ R}
```

---

#### (iv) Image of 6 under f

From the graph,

```text
f(6) = 5
```

Hence, the image of 6 under f is:

```text
5
```
