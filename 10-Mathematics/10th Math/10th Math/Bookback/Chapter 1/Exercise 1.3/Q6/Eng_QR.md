# Exercise 1.3

## Question 6:

A function f is defined by f(x) = 2x - 3
```text
(i) Find f(0) + f(1)
         ───────────
              2

(ii) Find x such that f(x) = 0.

(iii) Find x such that f(x) = x.

(iv) Find x such that f(x) = f(1-x).
```

---

## Solution:

Given,

```text
f(x) = 2x - 3
```

#### (i) To find:

```text
f(0) + f(1)
───────────
     2
```

```text
f(0) = 2(0) - 3
     = -3

f(1) = 2(1) - 3
     = -1
```

Therefore,

```text
f(0) + f(1)   -3 + (-1)
─────────── = ──────────
     2             2

             -4
           = ───
              2

           = -2
```

---

#### (ii) To find x such that f(x) = 0:

```text
2x - 3 = 0

2x = 3

x = 3
   ───
    2
```

---

#### (iii) To find x such that f(x) = x:

```text
2x - 3 = x

x = 3
```

---

#### (iv) To find x such that f(x) = f(1-x):

```text
f(x) = f(1-x)

2x - 3 = 2(1-x) - 3

2x - 3 = 2 - 2x - 3

2x + 2x = 2 - 3 + 3

4x = 2

x = 2
    ─
    4

x = 1
    ─
    2
```
