# பயிற்சி 1.3

## கேள்வி 6:

ஒரு சார்பு f ஆனது f(x) = 2x - 3 என வரையறுக்கப்பட்டுள்ளது.

```text
(i) f(0) + f(1)
    ───────────
         2

(ii) f(x) = 0 எனில், x ஐக் காண்க.

(iii) f(x) = x எனில், x ஐக் காண்க.

(iv) f(x) = f(1-x) எனில், x ஐக் காண்க.
```

---

## விடை:

கொடுக்கப்பட்டது,

```text
f(x) = 2x - 3
```

#### (i) கண்டுபிடிக்க:

```text
f(0) + f(1)
───────────
     2
```

```text
f(0) = 2(0) - 3
     = -3

f(1) = 2(1) - 3
     = -1
```

எனவே,

```text
f(0) + f(1)   -3 + (-1)
─────────── = ──────────
     2             2

             -4
           = ───
              2

           = -2
```

---

#### (ii) f(x) = 0 எனில், x ஐக் கண்டுபிடிக்க.

```text
2x - 3 = 0

2x = 3

x = 3
   ─
   2
```

---

#### (iii) f(x) = x எனில், x ஐக் கண்டுபிடிக்க.

```text
2x - 3 = x

2x - x = 3

x = 3
```

---

#### (iv) f(x) = f(1-x) எனில், x ஐக் கண்டுபிடிக்க.

```text
f(x) = f(1-x)

2x - 3 = 2(1-x) - 3

2x - 3 = 2 - 2x - 3

2x - 3 = -2x - 1

2x + 2x = -1 + 3

4x = 2

x = 2
    ─
    4

x = 1
    ─
    2
```
