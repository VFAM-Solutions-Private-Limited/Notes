# Exercise 1.3

## Question 8:

A function is defined by

```text
f(x) = 3 - 2x
```

Find x such that

```text
f(x)² = [f(x)]²
```

---

## Solution:

Given:

```text
f(x) = 3 - 2x
```

We have,

```text
f(x)² = [f(x)]²
```

Substituting the value of f(x):

```text
3 - 2x² = (3 - 2x)²
```

Expanding the square by (a-b)²:

```text
(a-b)² = a² - 2ab + b²

3 - 2x² = 3² - 2 x 3 x 2x + (2x)²
3 - 2x² = 9 - 12x + 4x²
```

Bringing all terms to one side:

```text
-4x² - 2x² + 12x + 3 - 9 = 0
-6x² + 12x - 6 = 0
```

Dividing throughout by -6:

```text
-6x² + 12x - 6
────   ───  ───
-6     -6   -6

x² - 2x + 1 = 0
```

Factoring:

```text
(x - 1)(x - 1) = 0
```

Hence,


```text
x = 1
```
