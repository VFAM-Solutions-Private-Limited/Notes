# பயிற்சி 1.3

## வினா 8:

f என்ற சார்பு

```text
f(x) = -3x²
```

என வரையறுக்கப்படுகிறது.

x-ஐக் காண்க; அவ்வாறு

```text
f(x²) = [f(x)]²
```

---

## விடை:

கொடுக்கப்பட்டது:

```text
f(x) = 3 - 2x
```

நமக்கு,

```text
f(x²) = [f(x)]²
```

f(x)-இன் மதிப்பைப் பிரதியிட்டால்,

```text
3 - 2x² = (3 - 2x)²
```

(a-b)² என்ற வாய்ப்பாட்டைப் பயன்படுத்தி விரிவாக்கினால்:

```text
(a-b)² = a² - 2ab + b²

3 - 2x² = 3² - 2 × 3 × 2x + (2x)²
3 - 2x² = 9 - 12x + 4x²
```

அனைத்து உறுப்புகளையும் ஒரு பக்கத்திற்குக் கொண்டுவந்தால்:

```text
-4x² - 2x² + 12x + 3 - 9 = 0
-6x² + 12x - 6 = 0
```

-6 ஆல் வகுத்தால்:

```text
-6x² + 12x - 6
────   ───  ───
-6     -6   -6

x² - 2x + 1 = 0
```

காரணிகளாகப் பிரித்தால்:

```text
(x - 1)(x - 1) = 0
```

எனவே,

```text
x = 1
```
