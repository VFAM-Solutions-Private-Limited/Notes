# Exercise 1.4

## Question 3:

Represent the function

```text
f = {(1,2), (2,2), (3,2), (4,3), (5,4)}
```

through

```text
(i) An arrow diagram

(ii) A table form

(iii) A graph
```

---

## Procedure to Solve (For Reference):

1. Write the given ordered pairs of the function.
2. Identify the domain and the corresponding range values.
3. Draw arrows from each element of the domain to its corresponding image.
4. Represent the function in a table.
5. Plot the ordered pairs on the Cartesian plane to obtain the graph.

---

## Solution:

#### Given:

```text
f = {(1,2), (2,2), (3,2), (4,3), (5,4)}
```

The ordered pairs are:

```text
(1,2)

(2,2)

(3,2)

(4,3)

(5,4)
```

The domain is

```text
A = {1,2,3,4,5}
```

The codomain is

```text
B = {2,3,4}
```
