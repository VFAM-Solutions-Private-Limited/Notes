# Exercise 1.4

## Question 6:

Let

```text
A = {1,2,3,4}

B = ℕ
```

Let

```text
f : A → B
```

be defined by

```text
f(x) = x³
```

Then,

```text
(i) Find the range of f.

(ii) Identify the type of function.
```

---

## Procedure to Solve (For Reference):

1. Write the given function.
2. Substitute each element of set **A** into the function.
3. Find the corresponding images.
4. Write the range of the function.
5. Check whether different elements have different images.
6. Identify the type of function.

---

## Solution:

#### Given:

```text
A = {1,2,3,4}

B = ℕ

f(x) = x³
```

Find the image of each element of **A**.

```text
f(1) = 1³

     = 1
```

```text
f(2) = 2³

     = 8
```

```text
f(3) = 3³

     = 27
```

```text
f(4) = 4³

     = 64
```

Hence,

| x | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| f(x) | 1 | 8 | 27 | 64 |

The range is

```text
Range(f)

= {1,8,27,64}
```

Since different elements of **A** have different images,

```text
f is one-one (Injective).
```

Since

```text
B = ℕ
```

contains many elements other than

```text
{1,8,27,64},
```

every element of **B** is not an image of an element of **A**.

Therefore,

```text
f is not onto.
```

---

## Final Answer:

```text
(i) Range(f) = {1,8,27,64}

(ii) f is a one-one (Injective) function,
     but not an onto function.
```
