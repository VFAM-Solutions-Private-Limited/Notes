# Example 1.8

## Question:

If

```text
X = {−5,1,3,4}
```

and

```text
(i)  R₁ = {(−5,a), (1,a), (3,b)}

(ii) R₂ = {(−5,b), (1,b), (3,a), (4,c)}

(iii) R₃ = {(−5,a), (1,a), (3,b), (4,c), (1,b)}
```

Determine whether each relation is a function or not.

## Procedure to solve (For reference):

1. Observe the ordered pairs or the arrow diagram.
2. Check whether every element of **X** has an image in **Y**.
3. Verify that each element of **X** has **only one** image.
4. If any element has no image or more than one image, the relation is **not** a function.

---

## Solution:



#### (i) Given:

```text
R₁ = {(−5,a), (1,a), (3,b)}
```

From the arrow diagram,

<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Example%201.7(i).jpeg" width="190">
</p>

The element **4 ∈ X** has no image in **Y**.

Therefore,

```text
R₁ is not a function.
```

---

#### (ii) Given:

```text
R₂ = {(−5,b), (1,b), (3,a), (4,c)}
```

From the arrow diagram,

<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Example%201.7(ii).jpeg" width="200">
</p>

Each element of **X** has exactly one image in **Y**.

Therefore,

```text
R₂ is a function.
```

---

#### (iii) Given:

```text
R₃ = {(−5,a), (1,a), (3,b), (4,c), (1,b)}
```

From the arrow diagram,

<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Example%201.7(iii).jpeg" width="180">
</p>

The element **1 ∈ X** has two images (**a** and **b**).

Therefore,

```text
R₃ is not a function.
```

---

## Final Answer:

```text
(i)  R₁ is not a function.

(ii) R₂ is a function.

(iii) R₃ is not a function.
```
