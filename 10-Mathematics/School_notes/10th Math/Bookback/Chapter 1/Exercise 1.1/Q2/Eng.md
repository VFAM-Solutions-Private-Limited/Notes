# Exercise 1.1

## Question 2:

Let A = {1,2,3} and B = {x | x is a prime number less than 10}.  
Find A x B and B x A.

---
## Procedure to solve (For reference):

1. Take one element from the first set.  
2. Pair it with every element of the second set.  
3. Repeat for all elements of the first set.  
4. Write the ordered pairs in the form (a,b).

---
## Solution:
#### Given:

A = {1,2,3}

To find B:

B = {x | x is a prime number less than 10}

Numbers less than 10 are:

1, 2, 3, 4, 5, 6, 7, 8, 9

Among these, prime numbers are:

2, 3, 5, 7

Therefore,

B = {2,3,5,7}

Hence, A = {1,2,3}, B = {2,3,5,7}

---
#### To find A x B and B x A:

```text
A x B = {1,2,3} x {2,3,5,7}

      = {(1,2), (1,3), (1,5), (1,7),(2,2), (2,3), (2,5), (2,7),(3,2), (3,3), (3,5), (3,7)}


B x A = {2,3,5,7} x {1,2,3}

      = {(2,1), (2,2), (2,3),(3,1), (3,2), (3,3),(5,1), (5,2), (5,3),(7,1), (7,2), (7,3)}
```
