# பயிற்சி 1.1

## கேள்வி 2:

A = {1,2,3} மற்றும் B = {x | x என்பது 10-ஐ விடச் சிறிய பகா எண் (Prime Number)} எனில்,  
A x B மற்றும் B x A ஆகியவற்றைக் காண்க.

---



## விடை:

#### கொடுக்கப்பட்டது:

A = {1,2,3}

B-ஐ நிர்ணயிக்க:

B = {x | x என்பது 10-க்கு குறைவான பகா எண் (Prime Number)}

10-க்கு குறைவான எண்கள்:

1, 2, 3, 4, 5, 6, 7, 8, 9

இவற்றில் பகா எண்கள் (Prime Numbers):

2, 3, 5, 7

எனவே,

B = {2,3,5,7}

ஆகவே,

A = {1,2,3}, B = {2,3,5,7}

---
#### A x B மற்றும் B x A:

```text
A x B = {1,2,3} x {2,3,5,7}

      = {(1,2), (1,3), (1,5), (1,7), (2,2), (2,3), (2,5), (2,7), (3,2), (3,3), (3,5), (3,7)}

B x A = {2,3,5,7} x {1,2,3}

      = {(2,1), (2,2), (2,3), (3,1), (3,2), (3,3), (5,1), (5,2), (5,3), (7,1), (7,2), (7,3)}
```
