# Exercise 1.1

## Question 3:

If B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}, find A and B.

---

## Procedure to solve (For reference):

1. In B x A, the first elements of each ordered pair belong to set B.  
2. In B x A, the second elements of each ordered pair belong to set A.  
3. Write each distinct element only once.  
4. Express the elements in set form.

---

## Solution:

#### Given:

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

We know that:

B x A = {(b,a) | b ∈ B and a ∈ A}

To find: A and B

This means:

- The first element of each ordered pair belongs to set B.  
- The second element of each ordered pair belongs to set A.

---
#### Answer:
```text
B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

    A = {3,4}

    B = {-2,0,3}
```
---
## For better understanding (optional):
```text
Step 1: To find set B

Take the first elements from all ordered pairs:

(-2,3) → -2  
(-2,4) → -2  
(0,3) → 0  
(0,4) → 0  
(3,3) → 3  
(3,4) → 3

Distinct first elements are: -2, 0, 3

Therefore, B = {-2,0,3}


Step 2: To find set A

Take the second elements from all ordered pairs:

(-2,3) → 3  
(-2,4) → 4  
(0,3) → 3  
(0,4) → 4  
(3,3) → 3  
(3,4) → 4

Distinct second elements are: 3, 4

Therefore, A = {3,4}


Step 3: Verification

Now verify:

B x A = {-2,0,3} x {3,4}

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

This matches the given set.

Final answer: 

    A = {3,4}

    B = {-2,0,3}
```
