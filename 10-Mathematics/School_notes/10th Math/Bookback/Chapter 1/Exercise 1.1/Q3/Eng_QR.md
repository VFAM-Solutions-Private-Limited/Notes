# Exercise 1.1

## Question 3:

If B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}, find A and B.

---


## Solution:

#### Given:

B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

We know that:

B x A = {(b,a) | b ∈ B and a ∈ A}

To find: A and B

This means:

- The first element of each ordered pair belongs to set B.  
- The second element of each ordered pair belongs to set A.

---
#### Answer:

```text
B x A = {(-2,3), (-2,4), (0,3), (0,4), (3,3), (3,4)}

    A = {3,4}

    B = {-2,0,3}
```
