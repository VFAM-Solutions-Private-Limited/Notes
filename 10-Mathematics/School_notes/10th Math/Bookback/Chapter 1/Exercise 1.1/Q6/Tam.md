# பயிற்சி 1.1


## கேள்வி 6

A = {x ∈ W | x < 2}, B = {x ∈ N | 1 < x ≤ 4} மற்றும் C = {3,5} எனில், கீழேக் கொடுக்கப்பட்டுள்ள சமன்பாடுகளைச் சரிபார்க்க.

(i) A × (B ∪ C) = (A × B) ∪ (A × C)<br>
(ii) A × (B ∩ C) = (A × B) ∩ (A × C)<br>
(iii) (A ∪ B) × C = (A × C) ∪ (B × C)

## தீர்க்கும் முறை (குறிப்புக்கு):

1. கொடுக்கப்பட்ட கணங்களை roster form-ல் எழுதுக.
2. தேவையான செயலைக் கண்டறிக (சேர்க்கை ∪ அல்லது வெட்டு ∩).
3. இடப்பக்கத்தின் (LHS) கார்டீசியன் பெருக்கலைக் கணக்கிடுக.
4. வலப்பக்கத்தின் (RHS) கார்டீசியன் பெருக்கல்களைத் தனித்தனியாகக் கணக்கிடுக.
5. RHS பெறுபேறுகளில் சேர்க்கை (∪) அல்லது வெட்டு (∩) செய்க.
6. அனைத்து வரிசை ஜோடிகளையும் (ordered pairs) மறுபடியும் வராமல் சரியாக எழுதுக.
7. LHS மற்றும் RHS கணங்களை ஒப்பிடுக.
8. இரு பக்கங்களிலும் ஒரே வரிசை ஜோடிகள் இருந்தால், சமன்பாடு சரிபார்க்கப்பட்டது.

---
## விடை:

#### கொடுக்கப்பட்டது:


- A = {x ∈ W | x < 2}

    W = முழு எண்கள் (Whole numbers) = {0,1,2,3,...}

    x < 2 ⇒ பெறப்படும் மதிப்புகள் 0 மற்றும் 1

    ∴ A = {0,1}

- B = {x ∈ N | 1 < x ≤ 4}

    N = இயல் எண்கள் (Natural numbers) = {1,2,3,4,...}

    1 < x ≤ 4 ⇒ பெறப்படும் மதிப்புகள் 2, 3, 4

    ∴ B = {2,3,4}



- C = {3,5}  (ஏற்கனவே பட்டியல் வடிவில் உள்ளது)

---

#### இறுதி கொடுக்கப்பட்ட தொகுதிகள்:

A = {0,1}  
B = {2,3,4}  
C = {3,5}

---

#### (i) A × (B ∪ C) = (A × B) ∪ (A × C)
```text
B ∪ C = {2,3,4,5}

A × (B ∪ C) = {(0,2), (0,3), (0,4), (0,5), (1,2), (1,3), (1,4), (1,5)} --->(1)

A × B = {(0,2), (0,3), (0,4), (1,2), (1,3), (1,4)}

A × C = {(0,3), (0,5), (1,3), (1,5)}

(A × B) ∪ (A × C) = {(0,2), (0,3), (0,4), (0,5), (1,2), (1,3), (1,4), (1,5)} --->(2)

எனவே, (1) மற்றும் (2)-இல் இருந்து,

A × (B ∪ C) = (A × B) ∪ (A × C)
```
---

#### (ii) A × (B ∩ C) = (A × B) ∩ (A × C)
```text
B ∩ C = {3}

A × (B ∩ C) = {(0,3), (1,3)} --->(3)

A × B = {(0,2), (0,3), (0,4), (1,2), (1,3), (1,4)}

A × C = {(0,3), (0,5), (1,3), (1,5)}

(A × B) ∩ (A × C) = {(0,3), (1,3)} --->(4)

எனவே, (3) மற்றும் (4)-இல் இருந்து,

A × (B ∩ C) = (A × B) ∩ (A × C)
```
---

#### (iii) (A ∪ B) × C = (A × C) ∪ (B × C)
```text
A ∪ B = {0,1,2,3,4}

(A ∪ B) × C = {(0,3), (0,5), (1,3), (1,5), (2,3), (2,5), (3,3), (3,5), (4,3), (4,5)} --->(5)

A × C = {(0,3), (0,5), (1,3), (1,5)}

B × C = {(2,3), (2,5), (3,3), (3,5), (4,3), (4,5)}

(A × C) ∪ (B × C) = {(0,3), (0,5), (1,3), (1,5), (2,3), (2,5), (3,3), (3,5), (4,3), (4,5)} --->(6)

எனவே, (5) மற்றும் (6)-இல் இருந்து,

(A ∪ B) × C = (A × C) ∪ (B × C)
```
---

## இறுதி விடை:

எனவே, மேலுள்ள மூன்று சமன்பாடுகளும் சரிபார்க்கப்பட்டன.
