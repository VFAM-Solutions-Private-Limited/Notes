# பயிற்சி 1.1

## கேள்வி 7

A என்பது 8-ஐ விடக் குறைவான இயல் எண்களின் கணம்,<br>
B என்பது 8-ஐ விடக் குறைவான பகா எண்களின் கணம்,<br>
C என்பது இரட்டை பகா எண்களின் கணம் எனில், கீழ்கண்டவற்றை 

சரிபார்க்கவும்:

(i) (A ∩ B) × C = (A × C) ∩ (B × C)<br>
(ii) A × (B − C) = (A × B) − (A × C)

---

## தீர்க்கும் முறை (குறிப்புக்கு):

1. கொடுக்கப்பட்ட கணங்களை **roster form (உறுப்புகள் வடிவம்)** ஆக மாற்றவும்.  
2. பயன்படுத்த வேண்டிய செயல்களை (∩, −, ×) கண்டறியவும்.  
3. **LHS (இடப்பக்கம்)**:
   - முதலில் set operation (∩ அல்லது −) செய்யவும்  
   - பின்னர் Cartesian product (×) செய்யவும்  
4. **RHS (வலப்பக்கம்)**:
   - முதலில் Cartesian product கணக்கிடவும்  
   - பின்னர் ∩ அல்லது − செயல்படுத்தவும்  
5. Cartesian product செய்யும்போது:
   - முதல் கணத்தின் ஒவ்வொரு உறுப்பையும் இரண்டாவது கணத்தின் ஒவ்வொரு உறுப்புடன் ஜோடியாக எழுதவும்  
6. அனைத்து ordered pairs-ஐ தெளிவாக எழுதவும்  
7. LHS மற்றும் RHS-ஐ ஒப்பிடவும்  
8. இரண்டும் சமமாக இருந்தால் → **சரிபார்க்கப்பட்டது (Verified)**  

---

## விடை:

#### கொடுக்கப்பட்டது:

- A = 8-ஐ விடக் குறைவான இயல் எண்கள்  
  ⇒ A = {1, 2, 3, 4, 5, 6, 7}

- B = 8-ஐ விடக் குறைவான பகா எண்கள்  
  ⇒ B = {2, 3, 5, 7}

- C = இரட்டை பகா எண்  
  ⇒ C = {2}

---

#### (i) (A ∩ B) × C = (A × C) ∩ (B × C)

```text
LHS:

A ∩ B = {2, 3, 5, 7}

(A ∩ B) × C
= {(2,2), (3,2), (5,2), (7,2)} --->(1)


RHS:

A × C
= {(1,2), (2,2), (3,2), (4,2), (5,2), (6,2), (7,2)}

B × C
= {(2,2), (3,2), (5,2), (7,2)}

(A × C) ∩ (B × C)
= {(2,2), (3,2), (5,2), (7,2)} --->(2)


LHS = RHS

(1) மற்றும் (2) இல் இருந்து,
(A ∩ B) × C = (A × C) ∩ (B × C)
```
---

#### (ii) A × (B − C) = (A × B) − (A × C)

```text
LHS:

B − C = {3,5,7}

A × (B − C)
= {(1,3), (1,5), (1,7),
   (2,3), (2,5), (2,7),
   (3,3), (3,5), (3,7),
   (4,3), (4,5), (4,7),
   (5,3), (5,5), (5,7),
   (6,3), (6,5), (6,7),
   (7,3), (7,5), (7,7)} --->(1)


RHS:

A × B
= {(1,2), (1,3), (1,5), (1,7),
   (2,2), (2,3), (2,5), (2,7),
   (3,2), (3,3), (3,5), (3,7),
   (4,2), (4,3), (4,5), (4,7),
   (5,2), (5,3), (5,5), (5,7),
   (6,2), (6,3), (6,5), (6,7),
   (7,2), (7,3), (7,5), (7,7)}

A × C
= {(1,2), (2,2), (3,2), (4,2), (5,2), (6,2), (7,2)}

(A × B) − (A × C)
= {(1,3), (1,5), (1,7),
   (2,3), (2,5), (2,7),
   (3,3), (3,5), (3,7),
   (4,3), (4,5), (4,7),
   (5,3), (5,5), (5,7),
   (6,3), (6,5), (6,7),
   (7,3), (7,5), (7,7)} --->(2)


LHS = RHS

(1) மற்றும் (2) இல் இருந்து,
A × (B − C) = (A × B) − (A × C)
```

மேலுள்ள இரண்டு சமன்பாடுகளும் சரிபார்க்கப்பட்டன (Verified).
