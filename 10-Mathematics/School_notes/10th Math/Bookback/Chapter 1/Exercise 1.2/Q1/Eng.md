# Exercise 1.2

## Question 1:

A = {1, 2, 3, 7}  
B = {3, 0, -1, 7}  

Which of the following are relation from A to B ?

(i) R1 = {(2,1), (7,1)} <br>
(ii) R2 = {(–1,1)} <br>
(iii) R3 = {(2,–1), (7,7), (1,3)} <br>
(iv) R4 = {(7,–1), (0,3), (3,3), (0,7)}

---

## Procedure to solve (For reference):

A relation from A to B must satisfy:  
- The first element of each ordered pair must belong to set A  
- The second element of each ordered pair must belong to set B  

---

#### (i) R1 = {(2,1), (7,1)}

Set diagram for R1 :

<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Exercise%201.2-1%28i%29.jpeg" width="250">

```text
- (2,1) : 2 ∈ A, 1 ∉ B 

- (7,1) :  7 ∈ A, 1 ∉ B  

2 and 7 cannot be related to 1.

Since, 1 ∉ B.

- R1 is not a relation.

```
---


#### (ii) R2 = {(-1,1)}

Set diagram for R2 :

```text
- (-1,1) : -1 ∉ A, 1 ∉ B 

1 cannot be related to 1.

Since, -1 ∉ A and 1 ∉ B.

- R2 is not a relation.

```

#### (iii) R3 = {(2,-1), (7,7), (1,3)}

Set diagram for R3 :


<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Exercise%201.2-1%28iii%29.jpeg" width="250">

```text
- (2,-1) : 2 ∈ A, -1 ∈ B 

- (7,7) : 7 ∈ A, 7 ∈ B  

- (1,3) : 1 ∈ A, 3 ∈ B 

Since, 2 is related to -1.
       7 is related to 7.
       1 is related to 3.

- R3 is a relation.
```
---

#### (iv) R4 = {(7,-1), (0,3), (3,3), (0,7)}
Set diagram for R4 :

<p align="center">

```text
- (7,-1) : 7 ∈ A, -1 ∈ B
- (0,3) : 0 ∉ A, 3 ∈ B
- (3,3) : 3 ∈ A, 3 ∈ B
- (0,7) : 0 ∉ A, 7 ∈ B

7 is related to -1.
3 is related to 3.

Since, 0 ∉ A
    -0 cannot be related to 3 and 7.

- R4 is not a relation.
```
---

## Final Answer:

The relation from A to B is:

**R3 only**
