# Exercise 1.2  

## கேள்வி 3:

R என்ற ஒரு உறவு<br>
R = { (x, y) | y = x + 3 , x ∈ {0, 1, 2, 3, 4, 5} }  
எனக் கொடுக்கப்பட்டுள்ளது.

 இதன் சார்பகத்தையும் வீச்சகத்தையும் காண்க.

---

## விடை:

#### கொடுக்கப்பட்டது:
x ∈ {0, 1, 2, 3, 4, 5}  
R: y = x + 3  

---

#### மதிப்புகளை காண்க:

```text
- x = 0 → y = 0 + 3 = 3 → (0, 3)  
- x = 1 → y = 1 + 3 = 4 → (1, 4)  
- x = 2 → y = 2 + 3 = 5 → (2, 5)  
- x = 3 → y = 3 + 3 = 6 → (3, 6)  
- x = 4 → y = 4 + 3 = 7 → (4, 7)  
- x = 5 → y = 5 + 3 = 8 → (5, 8)  
```
