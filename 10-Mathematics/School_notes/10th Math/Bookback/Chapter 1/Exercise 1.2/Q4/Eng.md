# Exercise 1.2  

## Question 4:

Represent each of the given relations by  
(a) an arrow diagram, (b) a graph and (c) a set in roster form, wherever possible.

(i) R = { (x, y) | x = 2y , x ∈ {2, 3, 4, 5}, y ∈ {1, 2, 3, 4} }<br>
(ii) R = { (x, y) | y = x + 3 , x, y are natural numbers < 10 }

---

## Procedure to solve (For reference):

1. Identify the given sets and the relation.  
2. Take each value from the given set (x or y).  
3. Apply the given relation formula.  
4. Find the corresponding value.  
5. Check whether the obtained value satisfies the given condition.  
6. If valid, form ordered pairs (x, y).  
7. Write all such pairs → R (Roster form).  
8. Draw the arrow diagram using the valid pairs.  
9. Plot the ordered pairs on a graph.  


---

## Solution

#### (i) Given:
x ∈ {2, 3, 4, 5}  
y ∈ {1, 2, 3, 4}  
R: x = 2y  

---

#### To find values:


```text
x = 2y → y = x/2

Substitute each value of x into y = x/2:

When x = 2y ⇒ y = x/2

When x = 2 ⇒ y = 2/2 = 1 ⇒ (2,1)

When x = 3 ⇒ y = 3/2 ⇒ (3,3/2)

When x = 4 ⇒ y = 4/2 = 2 ⇒ (4,2)

When x = 5 ⇒ y = 5/2 ⇒ (5,5/2)

R = {(2,1), (3,3/2), (4,2), (5,5/2)}
```

#### (ii) Given:
x, y are natural numbers < 10  
R: y = x + 3  

---

```text

When x = 1 ⇒ y = 4 ⇒ (1,4)

When x = 2 ⇒ y = 5 ⇒ (2,5)

When x = 3 ⇒ y = 6 ⇒ (3,6)

When x = 4 ⇒ y = 7 ⇒ (4,7)

When x = 5 ⇒ y = 8 ⇒ (5,8)

When x = 6 ⇒ y = 9 ⇒ (6,9)

When x = 7 ⇒ y = 10 ⇒ (7,10)

Whenv x = 8 ⇒ y = 11 ⇒ (8,11)

When x = 9 ⇒ y = 12 ⇒ (9,12)

R = {(1,4), (2,5), (3,6), (4,7), (5,8), (6,9), (7,10), (8,11), (9,12)}
```
