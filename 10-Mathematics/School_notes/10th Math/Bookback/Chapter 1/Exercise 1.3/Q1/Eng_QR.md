# Exercise 1.3

## Question 1:

Let  
R = { (x, y) | x, y ∈ N and y = 2x }  
be a relation on N.  

Find the domain, co-domain and range.  
Is this relation a function?

---

## Solution

#### Given:

x, y ∈ N  
Relation: y = 2x  

---

#### To find ordered pairs:

```text
x = 1 → y = 2 → (1, 2)

x = 2 → y = 4 → (2, 4)

x = 3 → y = 6 → (3, 6)

x = 4 → y = 8 → (4, 8)

x = 5 → y = 10 → (5, 10)

... and so on
```

#### Final Answer:
```text
Domain = {1, 2, 3, 4, 5, ...}
Co-domain = N = {1, 2, 3, 4, 5, ...}
Range = {2, 4, 6, 8, 10, ...}

Yes, this is a function because each x has exactly one y.
```
