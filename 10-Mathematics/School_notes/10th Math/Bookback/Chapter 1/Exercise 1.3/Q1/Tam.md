# பயிற்சி 1.3  

## கேள்வி 1:

R = { (x, y) | x, y ∈ N மற்றும் y = 2x }  
என்பது N மீது ஒரு உறவு ஆகும்.  

சார்பகம், துணை சார்பகம் மற்றும் வீச்சகம் காண்க.  
இந்த உறவு சார்பா?

---

## தீர்க்கும் முறை (குறிப்புக்கு):

1. x இன் மதிப்புகளை N (இயற்கை எண்கள்) இலிருந்து எடுக்கவும்.  
2. y = 2x என்ற உறவை பயன்படுத்தவும்.  
3. (x, y) என்ற வரிசைச் ஜோடிகளை உருவாக்கவும்.  
4. சார்பகம் → அனைத்து x மதிப்புகள்.  
5. துணை சார்பகம் → கொடுக்கப்பட்ட N.  
6. வீச்சகம் → கிடைக்கும் அனைத்து y மதிப்புகள்.  
7. ஒவ்வொரு x-க்கும் ஒரே y உள்ளதா என்பதை சரிபார்க்கவும்.  

---

## விடை:

#### கொடுக்கப்பட்டது:

x, y ∈ N  
உறவு: y = 2x  

---

#### வரிசைச் ஜோடிகள்:

```text
x = 1 → y = 2 → (1, 2)

x = 2 → y = 4 → (2, 4)

x = 3 → y = 6 → (3, 6)

x = 4 → y = 8 → (4, 8)

x = 5 → y = 10 → (5, 10)

... இவ்வாறு தொடரும்
```

#### இறுதி விடை:
```text
சார்பகம் = {1, 2, 3, 4, 5, ...}
துணை சார்பகம் = N = {1, 2, 3, 4, 5, ...}
வீச்சகம் = {2, 4, 6, 8, 10, ...}

ஆம், இது ஒரு சார்பாகும். ஏனெனில் ஒவ்வொரு x-க்கும் ஒரே ஒரு y மட்டுமே உள்ளது.
```
