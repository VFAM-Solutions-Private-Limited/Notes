# Exercise 1.3

## Question:

The data in the adjacent table depicts the length of a person's forehand and their corresponding height. Based on this data, a student finds a relationship between the height \(y\) and the forehand length \(x\) as:

```text
yᵃ = x + b
```

where a and b are constants.

| Length 'x' of forehand (cm) | Height 'y' (inches) |
|----------------------------|---------------------|
| 35 | 56 |
| 45 | 65 |
| 50 | 69.5 |
| 55 | 74 |

```text
(i) Check if this relation is a function.

(ii) Find a and b.

(iii) Find the height of a person whose forehand length is 40 cm.

(iv) Find the length of forehand of a person if the height is 53.3 inches.
```

---

## Procedure to solve (For reference):

1. Check whether each value of x has exactly one corresponding value of y.
2. Determine the values of a and b using the given data.
3. Form the relation between x and y.
4. Substitute the given values and find the required answers.

---

## Solution:

Given,

```text
yᵃ = x + b
```

### (i) Check if this relation is a function:

From the table,

```text
35 → 56
45 → 65
50 → 69.5
55 → 74
```

Each value of x corresponds to exactly one value of y.

Therefore,

```text
The relation is a function.
```

---

### (ii) Find a and b:

Observe the values:

```text
56 - 35 = 21

65 - 45 = 20

69.5 - 50 = 19.5

74 - 55 = 19
```

The difference is approximately constant.

Hence,

```text
a = 1
```

Therefore,

```text
y = x + b
```

Using the point (35, 56),

```text
56 = 35 + b

b = 56 - 35

b = 21
```

Therefore,

```text
a = 1

b = 21
```

Hence, the relation is

```text
y = x + 21
```

---

### (iii) Find the height of a person whose forehand length is 40 cm:

Given,

```text
x = 40
```

Using

```text
y = x + 21
```

```text
y = 40 + 21

y = 61
```

Therefore,

```text
Height = 61 inches
```

---

### (iv) Find the length of forehand of a person if the height is 53.3 inches:

Given,

```text
y = 53.3
```

Using

```text
y = x + 21
```

```text
53.3 = x + 21

x = 53.3 - 21

x = 32.3
```

Therefore,

```text
Length of forehand = 32.3 cm
```

---

## Final Answer:

```text
(i) The relation is a function.

(ii) a = 1, b = 21

(iii) Height of the person = 61 inches

(iv) Length of forehand = 32.3 cm
```
