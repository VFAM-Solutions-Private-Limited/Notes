# Exercise 1.3

## Question 2:

Let X = {3, 4, 6, 8}. Determine whether the relation 

R = { (x, f ( x ) ) |  x ∈ X, f ( x ) = x² + 1}

is a function from **X** to **N**?

---

## Solution:

Given:

```text
X = {3,4,6,8}

f(x) = x² + 1
```

---

Find the image of each element of X:

```text
f(3) = 3² + 1 = 10
f(4) = 4² + 1 = 17
f(6) = 6² + 1 = 37
f(8) = 8² + 1 = 65
```

Therefore,

```text
R = {(3,10), (4,17), (6,37), (8,65)}
```

---

### Domain:

```text
Domain(R) = {3,4,6,8}
```

### Range:

```text
Range(R) = {10,17,37,65}
```

---

### Verification:

```text
3 → 10
4 → 17
6 → 37
8 → 65
```

Each element of X has exactly one image in N.

---

## Final Answer:

```text
R = {(3,10), (4,17), (6,37), (8,65)}
```

Since every element of X is associated with exactly one natural number, **R is a function from X to N.**
