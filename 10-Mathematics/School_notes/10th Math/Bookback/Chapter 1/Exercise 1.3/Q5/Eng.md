# Exercise 1.3

## Question 5:

Let f(x) = 2x + 5. If x ≠ 0, find
```text
f(x+2) - f(2)
─────────────
      x
```

---

## Procedure to solve (For reference):

1. Find f(x+2) by substituting x+2 in the given function.
2. Find f(2) by substituting x=2
3. Substitute the values in the given expression.
4. Simplify the expression.

---

## Solution:

Given:

```text
f(x) = 2x + 5
```

To find f(x+2):

```text
f(x+2) = 2(x+2) + 5
       = 2x + 4 + 5
       = 2x + 9
```

To find f(2):

```text
f(2) = 2(2) + 5
     = 4 + 5
     = 9
```

Substitute in the given expression:

```text
f(x+2) - f(2)
─────────────
      x

= (2x+9) - 9
  ──────────
      x

= 2x
  ──
  x

= 2
```

Since x ≠ 0, division by x is valid.

---

## Final Answer:

```text
f(x+2) - f(2)
───────────── = 2
      x
```
