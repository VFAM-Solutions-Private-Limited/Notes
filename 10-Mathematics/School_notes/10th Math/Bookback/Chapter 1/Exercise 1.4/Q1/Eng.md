## Procedure to solve (For reference):

1. Apply the Vertical Line Test.
2. If a vertical line intersects the graph at more than one point, the graph is not a function.
3. If every vertical line intersects the graph at exactly one point, the graph is a function.

---

## Solution:

#### (i)

<p align="center">

Applying the Vertical Line Test:

```text
Some vertical lines intersect the graph at two points.
```

Therefore,

```text
The graph does NOT represent a function.
```

**Reason:** One value of x corresponds to two values of y.

#### (ii)
<p align="center">

Applying the Vertical Line Test:

```text
Every vertical line intersects the graph at only one point.
```

Therefore,

```text
The graph represents a function.
```

**Reason:** Each x-value has exactly one y-value.

Therefore,

The graph does NOT represent a function.

**Reason:** One x-value corresponds to more than one y-value.

| Graph | Function / Not a Function | Reason |
