# Exercise 1.4

## Question 5:

Show that the function

```text
f : ℕ → ℕ

f(m) = m² + m + 3
```

is a **one-one function**.

---

## Solution:

#### Given:

```text
f : ℕ → ℕ

f(m) = m² + m + 3
```

where

```text
ℕ = {1, 2, 3, 4, 5, ...}
```

Find the images of the first few natural numbers.

```text
f(1) = 1² + 1 + 3
     = 1 + 1 + 3
     = 5
```

```text
f(2) = 2² + 2 + 3
     = 4 + 2 + 3
     = 9
```

```text
f(3) = 3² + 3 + 3
     = 9 + 3 + 3
     = 15
```

```text
f(4) = 4² + 4 + 3
     = 16 + 4 + 3
     = 23
```

```text
f(5) = 5² + 5 + 3
     = 25 + 5 + 3
     = 33
```

Hence,

| m | 1 | 2 | 3 | 4 | 5 |
|------|---|---|---|---|---|
| f(m) | 5 | 9 | 15 | 23 | 33 |

The images are

```text
{5, 9, 15, 23, 33, ...}
```

Different elements of the domain have different images.

Therefore,

```text
The given function is one-one.
```


---

## Conclusion

```text
Hence, the function

f : ℕ → ℕ

f(m) = m² + m + 3

is a one-one function.
```
