# பயிற்சி 6.1

## வினா 4:

பின்வரும் முற்றொருமைகளை நிரூபிக்கவும்.

(i) sec⁶θ = tan⁶θ + 3tan²θsec²θ + 1

(ii) (sinθ + secθ)² + (cosθ + cosecθ)² = 1 + (secθ + cosecθ)²

---


## விடை:

#### (i)   sec⁶θ = tan⁶θ + 3tan²θsec²θ + 1

நிரூபிக்க:

```text
LHS = sec⁶θ

    = (sec²θ)³

    = (1 + tan²θ)³

    = 1 + 3tan²θ + 3tan⁴θ + tan⁶θ

    = 1 + 3tan²θ(1 + tan²θ) + tan⁶θ

    = 1 + 3tan²θ sec²θ + tan⁶θ

    = tan⁶θ + 3tan²θ sec²θ + 1

    = RHS

எனவே நிரூபிக்கப்பட்டது.
```

---

#### (ii)   (sinθ + secθ)² + (cosθ + cosecθ)² = 1 + (secθ + cosecθ)²

நிரூபிக்க:

```text
LHS = (sinθ + secθ)² + (cosθ + cosecθ)²

    = sin²θ + sec²θ + 2sinθsecθ + cos²θ + cosec²θ + 2cosθcosecθ

    = (sin²θ + cos²θ) + sec²θ + cosec²θ + 2tanθ + 2cotθ

    = 1 + sec²θ + cosec²θ + 2(tanθ + cotθ)

    = 1 + sec²θ + cosec²θ + 2(secθcosecθ)

    = 1 + (secθ + cosecθ)²

    = RHS

எனவே நிரூபிக்கப்பட்டது.
```
