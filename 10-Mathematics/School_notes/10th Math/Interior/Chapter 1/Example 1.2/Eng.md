# Example 1.2

## Question:

If

```text
A × B = {(3,2), (3,4), (5,2), (5,4)}
```

then find **A** and **B**.

---

## Procedure to solve (For reference):

1. Observe the first element of each ordered pair.
2. Write the distinct first elements to obtain set **A**.
3. Observe the second element of each ordered pair.
4. Write the distinct second elements to obtain set **B**.

---

## Solution:

To find: **A** and **B**

---

#### Given:

```text
A × B = {(3,2), (3,4), (5,2), (5,4)}
```

The first elements of the ordered pairs are

```text
3, 5
```

Therefore,

```text
A = {3,5}
```

The second elements of the ordered pairs are

```text
2, 4
```

Therefore,

```text
B = {2,4}
```

---

#### Verification:

```text
A × B = {3,5} × {2,4}

      = {(3,2), (3,4), (5,2), (5,4)}
```

This is the given Cartesian product.

---

## Final Answer:

```text
A = {3,5}

B = {2,4}
```
