# எடுத்துக்காட்டு 1.3

## கேள்வி:

```text
A = {x ∈ ℕ | 1 < x < 4},

B = {x ∈ W | 0 ≤ x < 2} மற்றும்

C = {x ∈ ℕ | x < 3} என்க.
```

பின்வருவனவற்றைச் சரிபார்க்க.

```text
(i) A × (B ∪ C) = (A × B) ∪ (A × C)

(ii) A × (B ∩ C) = (A × B) ∩ (A × C)
```

---

## தீர்க்கும் முறை (குறிப்பிற்காக):

1. கொடுக்கப்பட்டுள்ள கணங்களை பட்டியல் (Roster) வடிவில் எழுதுக.
2. B ∪ C மற்றும் B ∩ C ஆகியவற்றைக் காண்க.
3. தேவையான கார்ட்டீசியன் பெருக்கல்களைக் காண்க.
4. இடப்பக்கம் (LHS) மற்றும் வலப்பக்கம் (RHS)-ஐ தனித்தனியாகக் கணக்கிடுக.
5. இரண்டும் சமமாக இருப்பதைச் சரிபார்க்கவும்.

---

## விடை:

கொடுக்கப்பட்டுள்ளது,

```text
A = {2,3}

B = {0,1}

C = {1,2}
```

---

#### (i) சரிபார்க்க:

```text
A × (B ∪ C) = (A × B) ∪ (A × C)
```

```text
LHS:

B ∪ C

= {0,1} ∪ {1,2}

= {0,1,2}

A × (B ∪ C)

= {2,3} × {0,1,2}

= {(2,0), (2,1), (2,2), (3,0), (3,1), (3,2)}  ---->(1)
```

```text
RHS:

A × B

= {2,3} × {0,1}

= {(2,0), (2,1), (3,0), (3,1)}

A × C

= {2,3} × {1,2}

= {(2,1), (2,2), (3,1), (3,2)}

(A × B) ∪ (A × C)

= {(2,0), (2,1), (2,2), (3,0), (3,1), (3,2)}  ---->(2)
```

```text
LHS = RHS
```

எனவே, (1) மற்றும் (2)-இலிருந்து,

```text
A × (B ∪ C) = (A × B) ∪ (A × C)
```

என்பது சரிபார்க்கப்பட்டது.

---

#### (ii) சரிபார்க்க:

```text
A × (B ∩ C) = (A × B) ∩ (A × C)
```

```text
LHS:

B ∩ C

= {0,1} ∩ {1,2}

= {1}

A × (B ∩ C)

= {2,3} × {1}

= {(2,1), (3,1)}  ---->(1)
```

```text
RHS:

A × B

= {(2,0), (2,1), (3,0), (3,1)}

A × C

= {(2,1), (2,2), (3,1), (3,2)}

(A × B) ∩ (A × C)

= {(2,1), (3,1)}  ---->(2)
```

```text
LHS = RHS
```

எனவே, (1) மற்றும் (2)-இலிருந்து,

```text
A × (B ∩ C) = (A × B) ∩ (A × C)
```

என்பது சரிபார்க்கப்பட்டது.
