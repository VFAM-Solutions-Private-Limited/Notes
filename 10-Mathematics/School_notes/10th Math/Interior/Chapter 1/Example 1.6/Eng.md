# Example 1.6

## Question:

Let

```text
X = {1,2,3,4}

Y = {2,4,6,8,10}

R = {(1,2), (2,4), (3,6), (4,8)}
```

Show that **R** is a function and find its **domain, co-domain and range**.

  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Example%201.6.png" width="250">

---

## Procedure to solve (For reference):

1. Observe the given ordered pairs (or the arrow diagram).
2. Check whether every element of **X** is related to exactly one element of **Y**.
3. If each element of **X** has only one image in **Y**, then **R** is a function.
4. Write the domain, co-domain, and range of **R**.

---

## Solution:

#### Given:

```text
X = {1,2,3,4}

Y = {2,4,6,8,10}

R = {(1,2), (2,4), (3,6), (4,8)}
```

From the arrow diagram,

<p align="center">
  <img src="https://raw.githubusercontent.com/Jeevanya25Ravi/Vfam_Roteen/main/Roteen%20Maths/Example%201.6.png" width="250">
</p>


Each element of **X** has exactly one image in **Y**.

Therefore,

```text
R is a function.
```

---

### Domain

```text
Domain(R)

= {1,2,3,4}
```

---

### Co-domain

```text
Co-domain(R)

= {2,4,6,8,10}
```

---

### Range

```text
Range(R)

= {2,4,6,8}
```

---

## Final Answer:

```text
R is a function.

Domain(R) = {1,2,3,4}

Co-domain(R) = {2,4,6,8,10}

Range(R) = {2,4,6,8}
```
