# Example 1.7

## Question:

A relation

```text
f : X → Y
```

is defined by

```text
f(x) = x² − 2
```

where,

```text
X = {−2, −1, 0, 3}

Y = R
```

(i) List the elements of **f**.

(ii) Is **f** a function?

---

## Procedure to solve (For reference):

1. Substitute each element of **X** into the given relation \(f(x)=x^2-2\).
2. Find the corresponding image for every element.
3. Write the relation as a set of ordered pairs.
4. Check whether each element of the domain has exactly one image.
5. If yes, then **f** is a function.

---

## Solution:

#### Given:

```text
f(x) = x² − 2

X = {−2, −1, 0, 3}

Y = R
```

### (i) List the elements of f

```text
f(−2) = (−2)² − 2

      = 4 − 2

      = 2
```

```text
f(−1) = (−1)² − 2

      = 1 − 2

      = −1
```

```text
f(0) = 0² − 2

     = 0 − 2

     = −2
```

```text
f(3) = 3² − 2

     = 9 − 2

     = 7
```

Hence,

```text
f = { (−2,2), (−1,−1), (0,−2), (3,7) }
```

---

### (ii) Is f a function?

Each element of the domain **X** has exactly one image in **Y**.

Therefore,

```text
f is a function.
```

---

## Final Answer:

```text
(i) f = {(−2,2), (−1,−1), (0,−2), (3,7)}

(ii) f is a function.
```
